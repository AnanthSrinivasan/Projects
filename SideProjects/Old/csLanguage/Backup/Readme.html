Name: Language Example

 

Description:

 

This sample shows how the language control can be utilized

to define Language-specific syntax when saying system phrases.

This example should not be modified since only the bare mini-

mum Italian and French prompts are added.

 

 

Steps Required:

·         Enable the required events in the language control

·         Use VB code to examine the data, and reformat it as needed                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        

 

In this example, the number event has been enabled in French.

This is enabled on the Event Property Page of the Language

control.  With this enabled, a French_Number event will be

fired each time a number is to be played.

 

By associating our own logic in this event, we can redefine

how this a number is heard.  In this sample the

playing of the number 234 is redefined.
